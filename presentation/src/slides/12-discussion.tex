\begin{frame}{Analysis}
    \begin{itemize}
        \item \textbf{Math vs.\ AI Words:} Standard math formulas handle the exact calculations while the AI handles the written explanations. This keeps the numbers 100\% accurate and stops the AI from making up false statistics.
        \item \textbf{Fewer System Crashes:} Using LangGraph and LangSmith allows the system to catch errors and step backward to fix them instead of breaking the entire app if something goes wrong.
        \item \textbf{Processing Time:} Large files take a few minutes to pass through all 6 AI agents. The live status bar on the screen keeps the user updated so they are not left guessing.
    \end{itemize}
\end{frame}
